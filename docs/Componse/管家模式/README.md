# 管家模式 (Butler Mode)

AI Agent 管家模式是一个智能监控和汇报系统，用于实时跟踪主Agent的工作状态并向用户提供简洁的进度汇报。

## 核心功能

- **实时监控**：跟踪主Agent的所有关键操作
- **智能汇报**：根据TTS需求提供简洁状态更新
- **主动响应**：用户询问时提供详细状态信息
- **监控面板**：可视化展示工作进度和系统状态

## 设计原则

1. **简洁优先**：汇报内容适合TTS转换
2. **实时性**：及时反映主Agent状态变化
3. **智能化**：根据上下文调整汇报频率和详细程度
4. **可扩展**：支持新的监控指标和汇报方式
